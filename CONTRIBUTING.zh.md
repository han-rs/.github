# 贡献指南

## 联系方式

请在本仓库发起 *Discussion*.

## 贡献方式

- 提 *Issue* / 发起 *Discussion*
  - 如果您发现一篇值得翻译的好文章等, 请提一个 *Issue*.
  - 如果您发现有内容翻译有误, 请提一个 *Issue*.
  - 如果您有其他问题, 请发起一个 *Discussion*.
- 提交 PR
  - 如果您想对现有翻译内容进行修改, 请提交 PR.
  - 如果您已独自翻译一份新资料, 可以在个人仓库完成翻译工作后直接转移所有权, 我将邀请您成为组织成员.
